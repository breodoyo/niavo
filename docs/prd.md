# Product Requirements Document

## Product

Niavo

## Problem Statement

Organizations receive work from many different channels.

Examples include:

- Customer requests
- Internal tasks
- Repair jobs
- Purchase requests
- Support tickets

Managing these manually creates confusion and delays.

## Target Users

- Small businesses
- Schools
- Hospitals
- NGOs
- Startups
- Internal company teams

## MVP Features

### Authentication

- Register
- Login
- Logout

### Organizations

- Create organization
- Join organization

### Work Items

- Create
- Edit
- Assign
- Update status
- Delete

### Dashboard

- My Work
- Organization Work

## Non Goals

The MVP will NOT include:

- AI
- Automation
- Mobile App
- Billing
- Integrations
- Notifications

## Success Criteria

Users can manage work from creation to completion without using spreadsheets.
